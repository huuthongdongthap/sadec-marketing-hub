## 📋 SEO Template Used

```html
<!-- SEO Meta Tags -->
<title>[Page Name] | Sa Đéc Marketing Hub</title>
<meta name="description" content="[Description in Vietnamese]">
<meta name="keywords" content="[keywords]">
<meta name="robots" content="index, follow">
<meta name="author" content="Sa Đéc Marketing Hub">

<!-- Canonical URL -->
<link rel="canonical" href="https://sadecmarketinghub.com/[path]">

<!-- Open Graph Meta Tags -->
<meta property="og:title" content="[Page Name] | Sa Đéc Marketing Hub">
<meta property="og:description" content="[Description]">
<meta property="og:type" content="website">
<meta property="og:url" content="https://sadecmarketinghub.com/[path]">
<meta property="og:image" content="https://sadecmarketinghub.com/favicon.png">
<meta property="og:site_name" content="Sa Đéc Marketing Hub">
<meta property="og:locale" content="vi_VN">

<!-- Twitter Card -->
<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:title" content="[Page Name] | Sa Đéc Marketing Hub">
<meta name="twitter:description" content="[Description]">
<meta name="twitter:image" content="https://sadecmarketinghub.com/favicon.png">
<meta name="twitter:creator" content="@sadecmarketinghub">

<!-- Schema.org JSON-LD -->
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "WebPage",
  "name": "[Page Name]",
  "description": "[Description]",
  "url": "https://sadecmarketinghub.com/[path]",
  "image": "https://sadecmarketinghub.com/favicon.png",
  "publisher": {
    "@type": "Organization",
    "name": "Sa Đéc Marketing Hub",
    "url": "https://sadecmarketinghub.com",
    "logo": {
      "@type": "ImageObject",
      "url": "https://sadecmarketinghub.com/favicon.png"
    }
  },
  "inLanguage": "vi-VN"
}
</script>
```

## 🎯 SEO Best Practices Applied

1. **Title Tags:** 50-60 characters, includes page name + brand
2. **Meta Description:** 150-160 characters, Vietnamese language
3. **Open Graph:** Full OG tags for social sharing (Facebook, LinkedIn)
4. **Twitter Card:** Summary large image for Twitter sharing
5. **Canonical URLs:** Prevents duplicate content issues
6. **Schema.org:** Structured data for search engines
7. **Language:** vi-VN locale specified
8. **Robots:** index, follow for all pages
