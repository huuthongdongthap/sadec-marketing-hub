## Script Created

### scan-seo.py
**Purpose:** Quét và auto-add SEO metadata cho HTML files

**Features:**
- Scan toàn bộ HTML files
- Check 7 SEO tags (title, description, og:title, og:description, og:image, twitter:card, canonical)
- Auto-generate metadata từ filename/path
- Auto-add SEO block trước `</head>`
- Generate markdown report

**Usage:**
```bash
python3 scan-seo.py
# Output: reports/seo/seo-audit-YYYY-MM-DD.md
```
