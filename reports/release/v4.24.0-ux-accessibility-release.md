## ♿ Accessibility Components

### 1. Skip Link (`assets/js/components/skip-link.js`)

**Purpose:** Accessibility feature cho keyboard users để skip to main content

**Features:**
- Skip link element inserted at body start
- Alt+S keyboard shortcut
- Smooth scroll to main content
- ARIA labels và accessibility attributes
- Auto-creates main content landmark if missing

**Config:**
```javascript
{
  id: 'skip-link',
  href: '#main-content',
  keyboardShortcut: 'Alt+S'
}
```

---

### 2. Back to Top Button (`assets/js/components/back-to-top.js`)

**Purpose:** Scroll-to-top button xuất hiện sau khi scroll 300px

**Features:**
- Show/hide based on scroll position (300px threshold)
- Smooth scroll animation (500ms duration)
- Keyboard support (Enter/Space)
- Material Icons arrow_upward icon
- Bottom-right position
- RequestAnimationFrame for scroll performance

**Config:**
```javascript
{
  showAfterScroll: 300,
  scrollDuration: 500,
  position: 'bottom-right',
  icon: 'arrow_upward'
}
```

---

### 3. Reading Progress Bar (`assets/js/components/reading-progress.js`)

**Purpose:** Hiển thị reading progress bar cho pages với long content

**Features:**
- Progress bar for pages with content >500px
- Configurable position (top/bottom)
- Optional percentage label
- RequestAnimationFrame for scroll performance
- ARIA progressbar role

**Config:**
```javascript
{
  minHeight: 500,
  position: 'top',
  height: 4,
  color: 'primary',
  showPercentage: false
}
```
