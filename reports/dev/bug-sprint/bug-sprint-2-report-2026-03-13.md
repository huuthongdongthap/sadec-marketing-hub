## Bug Details

### Broken Imports - Remaining from Sprint #1

**Issue:** 11 portal HTML files still had wrong import paths after previous bug sprint

**Broken paths:**
- `/assets/js/admin-shared.js` → Should be `/assets/js/services/admin-shared.js`
- `/assets/js/ui-utils.js` → Should be `/assets/js/services/ui-utils.js`

**Files affected:**
```
portal/assets.html        portal/invoices.html
portal/payments.html      portal/approve.html
portal/login.html         portal/missions.html
portal/dashboard.html     portal/credits.html
portal/projects.html      portal/reports.html
portal/ocop-exporter.html
```

**Fix Applied:**
```bash
# Fixed admin-shared.js in portal/*.html
sed -i '' 's|src="/assets/js/admin-shared.js"|src="/assets/js/services/admin-shared.js"|g' portal/*.html

# Fixed ui-utils.js in affiliate/*.html portal/*.html
sed -i '' 's|src="/assets/js/ui-utils.js"|src="/assets/js/services/ui-utils.js"|g' affiliate/*.html portal/*.html
```
