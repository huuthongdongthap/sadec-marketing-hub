## 📋 Usage Examples

```javascript
// Basic toasts
Toast.success('Operation completed successfully!');
Toast.error('Something went wrong!');
Toast.warning('Please check your input');
Toast.info('New update available');

// Toast with title
Toast.show('File uploaded', { type: 'success', title: 'Upload Complete' });

// Toast with actions
Toast.show('New message received', {
    type: 'info',
    title: 'Notification',
    duration: 8000,
    actions: [
        { id: 'view', label: 'View', primary: true },
        { id: 'dismiss', label: 'Dismiss', primary: false }
    ]
});

// Clear all toasts
Toast.clear();
```
