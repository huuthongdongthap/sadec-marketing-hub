## 🐛 Bugs Fixed

### 1. Broken Import Paths

**Files:**
- `assets/js/dashboard-client.js` (lines 9-11)
- `assets/js/finance-client.js` (lines 9-11)

**Before:**
```javascript
import { ApiClientBase } from '../shared/api-client.js';
import { formatCurrencyVN } from '../services/core-utils.js';
import { invoices, clients } from '../supabase.js';
```

**After:**
```javascript
import { ApiClientBase } from './shared/api-client.js';
import { formatCurrencyVN } from './services/core-utils.js';
import { invoices, clients } from './supabase.js';
```

**Root Cause:** Relative paths `../` were pointing outside `assets/js/` directory.

---

### 2. Toast Undefined Reference

**File:** `assets/js/features/quick-actions.js` (line 302)

**Fix:**
```javascript
// Added Toast reference before usage
const Toast = window.SadecToast || window.Toast;
Toast?.info('Đang chuẩn bị xuất dữ liệu...');
```

---

### 3. Console.log in Production

**Files:**
- `assets/js/features/data-export.js`
- `assets/js/features/user-preferences.js`
- `assets/js/features/quick-actions.js`

**Fix:** Added `_debug()` helper:
```javascript
const _debug = (message) => {
    if (typeof process !== 'undefined' && process.env?.NODE_ENV === 'development') {
        console.log(message);
    }
};
```
