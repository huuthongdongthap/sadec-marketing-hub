# Release Notes v1.1.0 — 2026-03-13

## Tổng quan

Release v1.1.0 tập trung vào việc **chuẩn hóa đường dẫn assets** và **cải thiện độ tin cậy** khi deploy production cho Sadec Marketing Hub.

---

## 🎯 Mục tiêu

- Sửa lỗi broken links khi deploy production
- Chuẩn hóa đường dẫn assets từ root domain
- Tăng cường công cụ audit và kiểm thử

---

## 📝 Thay đổi chính

### 1. Chuẩn hóa Absolute Paths (Breaking Change)

**Trước:**
```html
<link rel="stylesheet" href="../assets/css/m3-agency.css">
<script type="module" src="../auth.js"></script>
```

**Sau:**
```html
<link rel="stylesheet" href="/assets/css/m3-agency.css">
<script type="module" src="/auth.js"></script>
```

**Phạm vi áp dụng:**
- `admin/` — 45 files
- `affiliate/` — 8 files
- `portal/` — 20 files
- `auth/` — 1 file
- `lp.html` — 1 file

**Tổng cộng:** 73 files HTML được cập nhật

## 🔧 Technical Details

### Path Patterns Updated

| Pattern | Count |
|---------|-------|
| `../assets/` → `/assets/` | ~200 |
| `../auth.js` → `/auth.js` | ~70 |
| `../supabase-config.js` → `/supabase-config.js` | ~70 |
| `../favicon.png` → `/favicon.png` | ~70 |

### Benefits

1. **Reliability**: Assets resolve từ root domain, không phụ thuộc vào depth của file
2. **Deploy Safety**: Không còn lỗi 404 khi deploy trên Cloudflare Pages
3. **Maintainability**: Dễ dàng refactor thư mục mà không broken links
4. **Performance**: Browser cache assets tốt hơn với canonical paths

## ⚠️ Migration Notes

### Nếu bạn đang fork/clone repo:

