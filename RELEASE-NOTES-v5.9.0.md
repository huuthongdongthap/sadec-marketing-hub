## 🔧 Technical Details

### Utility Architecture

```
core/
├── core-utils.js         # Central export (single source of truth)
├── enhanced-utils.js     # UI components (Toast, ThemeManager, ScrollProgress)
└── utils.js              # Legacy compatibility layer (re-export)

shared/
├── format-utils.js       # Format functions (currency, date, number, string)
├── dom-utils.js          # DOM utilities ($, $$, on, off, delegate)
├── api-client.js         # API client base class with caching
├── modal-utils.js        # Modal utilities
└── guard-utils.js        # Guard utilities
```

### Import Pattern

```javascript
// Recommended: Import from core-utils (single source)
import { formatCurrency, Toast, ThemeManager } from './core/core-utils.js';

// Alternative: Import from shared (specific utilities)
import { formatCurrency } from './shared/format-utils.js';
import { $, $$, on, delegate } from './shared/dom-utils.js';
```
